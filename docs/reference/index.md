# Reference

Referenční sekce obsahuje přehledy, tabulky a úplné seznamy nastavení, parametrů a hodnot dostupných v Competent. Materiál je uspořádán tak, abyste konkrétní informaci rychle dohledali — například jaké hodnoty přijímá určité pole, jaké jsou dostupné volby v číselníku nebo jaká oprávnění má daná role. Pokud hledáte postup, jak nastavení použít, nahlédněte do sekce [Návody](../how-to/index.md).
