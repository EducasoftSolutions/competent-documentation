# Tutoriály

Tutoriály jsou průvodci určené pro ty, kdo s Competent teprve začínají. Každý tutoriál Vás provede ucelenými kroky od začátku do konce tak, abyste po jeho dokončení měli konkrétní výsledek a jistotu, že systém ovládáte. Nejde o popis všech možností — cílem je, abyste se rychle zorientovali a mohli začít pracovat.
