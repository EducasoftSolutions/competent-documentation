# Koncepty

Koncepty vysvětlují, jak Competent přemýšlí — jaká je logika procesů, jak spolu jednotlivé části systému souvisejí a proč je určitá funkce navržena tak, jak je. Čtení konceptů není nutné pro každodenní práci, ale pomůže Vám lépe porozumět chování systému, efektivněji troubleshootovat a správně nastavit složitější konfigurace. Konkrétní postupy najdete v sekci [Návody](../how-to/).
