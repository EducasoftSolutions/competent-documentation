# Slovník

Tato stránka obsahuje přehled základních pojmů používaných v Competent. Každý termín naleznete i v příslušné části dokumentace, na kterou odkazuje sloupec **Detail**.

| Termín                                     | Definice                                                                                                                                                             | Detail                                                                                                                                           |
| ------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| Aktivita                                   | Nejmenší splnitelná jednotka systému, přiřaditelná uživatelům; definována názvem, subtypem, schématem a obsahem.                                                     | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/aktivita/index.md)                             |
| Automaticky prodloužený přístup k aktivitě | Nedokončené přístupy se v datum konce neukončí, ale prodlouží o nastavený čas; opakuje se do vyčerpání počtu prodloužení.                                            |                                                                                                                                                  |
| Beztermínové schéma aktivity               | Schéma pro aktivity plněné samostatně, nezávislé na datu a místě konání (online kurzy, periodická školení).                                                          | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/schemata-aktivity/index.md)                    |
| Certifikát                                 | Listina potvrzující splnění aktivity, kterou Competent generuje z ODT šablony.                                                                                       | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/certifikat/index.md)                           |
| Datové typy                                | Typy hodnot, které mohou mít parametry objektů a subtypů: text, číslo, ano/ne, datum, datum a čas, e-mail, heslo, odkaz na objekt, výběr z možností a identifikátor. |                                                                                                                                                  |
| Hlavní objekt                              | Objekty s pevnými hlavními parametry a volitelnými subtypy: Aktivita, Sada, Termínová sada, Hodnocení, Uživatel, Uživatelská skupina, Kvalifikace.                   |                                                                                                                                                  |
| Hodnocení                                  | Hlavní objekt sdílející datový model s aktivitou, ale s jiným využitím: nevyžaduje plnění, uživatel je hodnocen.                                                     |                                                                                                                                                  |
| Hodnotitel                                 | Uživatel, který má přiřazeno hodnocení přidělených uživatelů; pracuje přes záložku K hodnocení.                                                                      |                                                                                                                                                  |
| Import/Export (xlsx)                       | Hromadné vytváření a export objektů přes tabulky xlsx s povinnou strukturou.                                                                                         |                                                                                                                                                  |
| Jazyková podpora                           | Lokalizace systému do více jazyků na úrovni serveru, rozhraní i názvů vybraných objektů.                                                                             |                                                                                                                                                  |
| Místo                                      | Entita popisující fyzickou lokaci (místo konání, adresa, bydliště uživatele).                                                                                        |                                                                                                                                                  |
| Notifikace                                 | Automatické e-mailové zprávy zasílané definovaným uživatelům při určité události.                                                                                    | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/emailove-notifikace/index.md)                  |
| Oprávnění                                  | Povolení uživatele využívat funkce systému (globální) nebo nakládat s objekty (objektová).                                                                           | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/role/index.md)                                 |
| Parametr                                   | Vlastnost objektu; hlavní (pevné) nebo vedlejší (přidávané přes subtypy).                                                                                            |                                                                                                                                                  |
| Periodické nastavení                       | Opakované automatické obnovování přístupu k aktivitě nebo sadě po jeho vypršení.                                                                                     | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/periodicke-nastaveni/index.md)                 |
| Podmínky aktivit a kvalifikací             | Podmínění spuštění aktivity nebo platnosti kvalifikace splněním jiných aktivit či kvalifikací.                                                                       | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/podminky-pristupu/index.md)                    |
| Pokusy uživatele                           | Soubor jednotlivých pokusů uživatele o splnění aktivity; každé spuštění tvoří nový pokus.                                                                            | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/pokusy-uzivatele/index.md)                     |
| Proměnné                                   | Zástupné symboly v e-mailových šablonách, automaticky nahrazené daty ze systému.                                                                                     | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/emailove-notifikace/index.md)                  |
| Přihlášení jako jiný uživatel              | Administrační nástroj umožňující procházet systém z pohledu jiného uživatele; vázán na zvláštní oprávnění.                                                           |                                                                                                                                                  |
| Přiřazení aktivity uživateli (přístup)     | Objekt vazby mezi uživatelem a aktivitou; popisuje jak, kdy a v jakém stavu uživatel aktivitu plní.                                                                  | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/prirazeni-aktivity-uzivateli-pristup/index.md) |
| Přiřazení dle skupin                       | Pravidlo automatického přiřazování aktivit na základě členství v uživatelských skupinách.                                                                            | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/prirazeni-dle-skupin/index.md)                 |
| Příkazy serveru                            | Sada příkazů, jimiž oprávněný administrátor udržuje systém v chodu.                                                                                                  |                                                                                                                                                  |
| Recenze                                    | Uživatelské hodnocení aktivity (hvězdičky, slovní hodnocení); může být povinné.                                                                                      |                                                                                                                                                  |
| Role                                       | Soubor oprávnění; uživatel je nositelem role vůči objektu nebo globálně.                                                                                             | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/role/index.md)                                 |
| Sada                                       | Seskupení aktivit, sad a termínových sad do celku přiřaditelného jako celek.                                                                                         | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/sada/index.md)                                 |
| Subtyp                                     | Detailní rozlišení hlavního objektu; každý objekt musí mít subtyp, po vytvoření neměnný.                                                                             | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/subtypy-aktivit/index.md)                      |
| Termínová sada                             | Kombinace sady a termínové aktivity; lze do ní přiřazovat aktivity a vytvářet termíny.                                                                               | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/terminova-sada/index.md)                       |
| Typ hodnocení                              | Určuje způsob vyhodnocení splnění aktivity (Textové, Body, Prošel/Neprošel, Procentuální, Formulář, Složené).                                                        |                                                                                                                                                  |
| Typ spouštění                              | Určuje typ obsahu aktivity a co se stane při spuštění (Žádný, Url, SCORM, iTrivio, PDF, Složené).                                                                    |                                                                                                                                                  |
| Účastník                                   | Společný jmenovatel interních rolí přiřazených uživateli jako členovi skupiny, kurzistovi nebo osobě na pozici.                                                      | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/role/index.md)                                 |
| Uživatel                                   | Osoba zavedená do systému; má přiřazené parametry, role a přístupy k aktivitám.                                                                                      | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/uzivatel/index.md)                             |
| Uživatelská práva                          | Akce povolené uživateli; každá akce má objekt oprávnění, oprávnění jsou organizována do rolí.                                                                        | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/role/index.md)                                 |
| Uživatelská skupina                        | Hlavní objekt pro klasifikaci uživatelů; umožňuje filtrování, hromadné přiřazení oprávnění i zpráv.                                                                  | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/skupina/index.md)                              |
| Vlastník                                   | Systémová role automaticky přiřazená tvůrci hlavního objektu; opravňuje k plné kontrole objektu.                                                                     | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/role/index.md)                                 |
| Vlastnosti aplikace                        | Soustava nastavení prostředí; část v konfiguračních souborech, část měnitelná za běhu.                                                                               |                                                                                                                                                  |
| Výběr uživatelů                            | Panel pro výběr uživatelů při přiřazování k aktivitám a hodnocení.                                                                                                   |                                                                                                                                                  |
| Zasílání e-mailů                           | Hlavní komunikační nástroj; systém odesílá notifikace i zprávy přes vlastní frontu a externí SMTP.                                                                   | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/emailove-notifikace/index.md)                  |
| Štítky                                     | Klasifikace aktivit do skupin pro snazší orientaci; přiřazují se v detailu aktivity.                                                                                 | → [Více](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/stitky/index.md)                               |
