# Záložka Místa

Záložka **Místa** je součástí obrazovky **Nastavení** a slouží ke správě sdíleného číselníku míst v systému Competent. Místo představuje adresní záznam, který lze přiřazovat objektům v systému (například aktivitám). Správou míst na tomto místě zajistíte konzistentní adresní údaje napříč celou instalací.

______________________________________________________________________

## Horní lišta

V horní části záložky se nachází ovládací prvky pro řazení a přidávání míst.

| Prvek                            | Popis                                                   |
| -------------------------------- | ------------------------------------------------------- |
| **Seřadit podle**                | Rozbalovací menu pro výběr řazení seznamu míst.         |
| **Tlačítko s ikonou +** (vpravo) | Otevře modál **Nové místo** pro přidání nového záznamu. |

### Možnosti řazení

| Hodnota   | Popis                                     |
| --------- | ----------------------------------------- |
| **Názvu** | Místa jsou seřazena abecedně podle názvu. |
| **Města** | Místa jsou seřazena abecedně podle města. |
| **Země**  | Místa jsou seřazena abecedně podle státu. |

______________________________________________________________________

## Seznam míst

Pod horní lištou se zobrazuje seznam míst seřazený podle vybraného atributu. U každého záznamu jsou uvedeny jeho název a adresní údaje. Dlouhý záznam lze rozbalit rozbalovacím prvkem na konci řádku.

| Prvek                     | Popis                                                              |
| ------------------------- | ------------------------------------------------------------------ |
| **Řádek místa**           | Kliknutím na záznam se otevře modál s formulářem pro úpravu místa. |
| **Ikona koše** (na řádku) | Vyvolá potvrzovací dialog pro smazání záznamu.                     |

Smazání přiřazeného místa

Místo, které je přiřazeno nějakému objektu v systému, **nelze smazat**. Systém v takovém případě zobrazí chybovou hlášku. Před smazáním je nutné ověřit, že místo není nikde přiřazeno.

______________________________________________________________________

## Modál Nové místo

Modál **Nové místo** se otevře kliknutím na tlačítko s ikonou + v horní liště záložky. Tentýž modál se používá i pro úpravu existujícího záznamu.

| Pole                | Popis                                                                               |
| ------------------- | ----------------------------------------------------------------------------------- |
| **Název**           | Název místa. Povinné pole. Pokud zůstane prázdné, uloží se předvyplněný šedý název. |
| **Popis**           | Volitelný popis místa.                                                              |
| **Stát**            | Stát nebo země, ve které se místo nachází.                                          |
| **Město**           | Město. Zobrazuje se na stejném řádku jako PSČ.                                      |
| **PSČ**             | Poštovní směrovací číslo.                                                           |
| **Ulice**           | Název ulice. Zobrazuje se na stejném řádku jako čísla popisné a orientační.         |
| **Č. orientační**   | Část adresy místa.                                                                  |
| **Č. popisné**      | Část adresy místa.                                                                  |
| **Patro**           | Patro budovy.                                                                       |
| **Místnost**        | Označení místnosti.                                                                 |
| **Překladový klíč** | Rezervováno pro budoucí vícejazyčné názvy míst. Viz poznámka níže.                  |

### Tlačítka modálu

| Tlačítko           | Popis                         |
| ------------------ | ----------------------------- |
| **Zrušit**         | Zavře modál bez uložení změn. |
| **Vytvořit místo** | Uloží nový záznam místa.      |

Překladový klíč

Pole **Překladový klíč** je rezervováno pro budoucí podporu vícejazyčných názvů míst. V současné době se v systému nepoužívá.
