# Obrazovka Nastavení

Obrazovka **Nastavení** slouží ke konfiguraci systému Competent podle potřeb vaší organizace. Nastavení škálují od jednoduché správy štítků a míst až po řízení komplexních workflow. Obrazovka je organizována do záložek – každá záložka spravuje určitou část konfigurace systému.

Přístup jen pro proškolené administrátory

Tato sekce by měla být přístupná jen několika málo proškoleným uživatelům, kteří dobře znají fungování systému Competent. Nesprávná manipulace s nastavením může nenávratně poškodit spravovaná data.

______________________________________________________________________

## Přístup k obrazovce

Vstup do obrazovky Nastavení a do jednotlivých záložek je řízen oprávněními. Různé záložky mohou vyžadovat různá oprávnění.

______________________________________________________________________

## Záložky obrazovky Nastavení

Obrazovka Nastavení obsahuje panel s 12 záložkami. Přehled záložek v pořadí, v jakém se zobrazují v rozhraní:

| Záložka                                                                                                                                                 | Popis                                                                                                                                                                                                                                                                                                                         |
| ------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**Místa**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/tab-mista/index.md)                               | Správa sdíleného číselníku míst (adres), která lze přiřazovat objektům v systému.                                                                                                                                                                                                                                             |
| [**Štítky**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/tab-stitky/index.md)                             | Správa štítků sloužících ke klasifikaci aktivit.                                                                                                                                                                                                                                                                              |
| **Menu**                                                                                                                                                | Samostatná dokumentace se připravuje.                                                                                                                                                                                                                                                                                         |
| **Role**                                                                                                                                                | Správa rolí a oprávnění. Viz také [Přehled rolí a oprávnění](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/prehled-roli-a-opravneni/index.md) a [Role: model a principy](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/concepts/role/index.md). |
| [**Subtypy**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/tab-subtypy/index.md)                           | Správa subtypů aktivit a dalších objektů.                                                                                                                                                                                                                                                                                     |
| [**Parametry**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/tab-parametry/index.md)                       | Správa parametrů hlavních objektů Competentu.                                                                                                                                                                                                                                                                                 |
| [**Přiřazení dle skupin**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/tab-prirazeni-dle-skupin/index.md) | Pravidla automatického přiřazování objektů uživatelům na základě příslušnosti ke skupinám.                                                                                                                                                                                                                                    |
| [**Šablony e-mailů**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/obrazovka-sablony-emailu/index.md)      | Správa skupin e-mailových šablon.                                                                                                                                                                                                                                                                                             |
| [**Notifikace**](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/tab-notifikace/index.md)                     | Správa globálních notifikací (událost, šablona, příjemci).                                                                                                                                                                                                                                                                    |
| **Koš**                                                                                                                                                 | Obnova a trvalé odstranění dříve smazaných objektů.                                                                                                                                                                                                                                                                           |
| **Workflows**                                                                                                                                           | Samostatná dokumentace se připravuje.                                                                                                                                                                                                                                                                                         |
| **Superadmin**                                                                                                                                          | Samostatná dokumentace se připravuje.                                                                                                                                                                                                                                                                                         |

Záložky Menu, Workflows a Superadmin slouží k pokročilejší konfiguraci systému; jejich samostatná dokumentace se připravuje.
