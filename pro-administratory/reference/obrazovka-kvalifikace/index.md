# Obrazovka Kvalifikace

Obrazovka Kvalifikace zobrazuje přehled definovaných kvalifikací a umožňuje založit novou kvalifikaci. Kliknutí na kartu v seznamu otevře detail dané kvalifikace, jehož záložky popisuje samostatná stránka. Model kvalifikace (stavy, platnost, podmínky) vysvětluje koncept Kvalifikace.

______________________________________________________________________

## Prvky obrazovky

| Prvek                                    | Popis                                                                                          |
| ---------------------------------------- | ---------------------------------------------------------------------------------------------- |
| **Seznam kvalifikací**                   | Plochý seznam karet; podrobnosti viz [Seznam kvalifikací](#seznam-kvalifikaci) níže.           |
| **Tlačítko +**                           | Otevře modál **Nová kvalifikace**; viz [Modál Nová kvalifikace](#modal-nova-kvalifikace) níže. |
| **Ikona koše** (po najetí myší na kartu) | Spustí smazání kvalifikace; viz [Mazání kvalifikace](#mazani-kvalifikace) níže.                |

______________________________________________________________________

## Seznam kvalifikací

Obrazovka Kvalifikace obsahuje plochý seznam karet, nikoli stromovou strukturu. Každá karta zobrazuje **jméno** kvalifikace a její **subtyp**.

Seznam nemá sloupce, filtry ani vyhledávání. Kliknutí na kartu otevře [detail kvalifikace](https://educasoftsolutions.github.io/competent-documentation/pro-administratory/reference/detail-kvalifikace/index.md).

______________________________________________________________________

## Modál Nová kvalifikace

Tlačítko **+** v levé horní části obrazovky otevře modál **Nová kvalifikace**.

Modál obsahuje tato pole:

| Pole       | Popis                                    |
| ---------- | ---------------------------------------- |
| **Subtyp** | Výchozí hodnotou je „Kvalifikace".       |
| **Jméno**  | Textové pole pro jméno nové kvalifikace. |

Potvrzení modálu zajišťuje tlačítko **Vytvořit**, zrušení tlačítko **Zrušit**.

______________________________________________________________________

## Mazání kvalifikace

Najetí myší na kartu v seznamu zobrazí ikonu koše. Její aktivace spustí potvrzovací dialog; po jeho potvrzení se kvalifikace smaže.

______________________________________________________________________

## Dostupnost obrazovky

Obrazovka Kvalifikace je dostupná administrátorům.
